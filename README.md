# BorrowAtoZ

BorrowAtoZ is an **advanced, production-grade rental marketplace** built using the **MERN + Next.js stack**, designed to mirror real-world platforms like Airbnb-style systems for **items and services**.  
The project emphasizes **scalable backend architecture, real-time communication, and clean domain modeling**.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB, Mongoose  
- **Frontend:** Next.js, Redux Toolkit, Tailwind CSS  
- **Auth & Security:** JWT, bcrypt, role-based access control  
- **Real-Time:** **Socket.IO** (live bookings, notifications, status updates)  
- **Other:** Pagination, advanced filtering & sorting, image uploads

## Core Features
- Secure authentication with email/phone verification
- Create, update, pause, and manage rental listings
- Item & service separation with category mapping
- Real-time notifications using Socket.IO
- Production-grade REST APIs with validation & access control
- Scalable querying (search, filters, pagination)
- Image management with upload limits
- Admin-controlled category management
